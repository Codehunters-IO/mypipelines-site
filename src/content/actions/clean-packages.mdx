## Uso

```yaml title=".github/workflows/clean-packages.yml"
- name: Clean old Maven package versions
  uses: ./.github/actions/packages
  with:
    token: ${{ secrets.GH_PACKAGES_TOKEN }}
    owner: ${{ github.repository_owner }}
    ownerType: user
    packageType: maven
    versions: 5
    packageNames: >-
      io.codehunters.commons.commons-util,
      io.codehunters.commons.commons-dto,
      io.codehunters.commons.commons-web
```

## Comportamiento

- Las versiones se listan de mĆ”s nueva a mĆ”s vieja; se conservan las primeras
  `versions`, el resto se borra.
- Paquetes con menos versiones que `versions` se saltan.
- El borrado corre secuencial por paquete; un fallo aborta la corrida.
